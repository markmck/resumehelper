   - Use Drizzle `db.select().from(table)`, `db.insert(table).values(...)`, `db.update(table).set(...)`, `db.delete(table).where(...)` patterns

   Entity-specific details:
   - **education.ts**: list returns all rows. create accepts `{ institution, area?, studyType?, startDate?, endDate?, score?, courses?: string[] }`. Store courses as JSON string `JSON.stringify(courses ?? [])`. Return parsed courses in list results `JSON.parse(row.courses)`.
   - **volunteer.ts**: list returns all rows. create accepts `{ organization, position?, startDate?, endDate?, summary?, highlights?: string[] }`. Store highlights as JSON string. Return parsed highlights in list.
   - **awards.ts**: Simple CRUD. create accepts `{ title, date?, awarder?, summary? }`.
   - **publications.ts**: Simple CRUD. create accepts `{ name, publisher?, releaseDate?, url?, summary? }`.
   - **languages.ts**: Simple CRUD. create accepts `{ language, fluency? }`.
   - **interests.ts**: create accepts `{ name, keywords?: string[] }`. Store keywords as JSON string.
   - **references.ts**: Simple CRUD. create accepts `{ name, reference? }`. IPC channel names: `references:list`, `references:create`, etc.
