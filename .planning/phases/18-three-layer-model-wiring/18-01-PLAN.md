    3. After the existing code builds `jobsWithBullets` array (the jobs with their bullets attached), add this block before the return statement:

    ```typescript
    if (analysisId != null) {
      const overrideRows = db.select({
        bulletId: analysisBulletOverrides.bulletId,
        overrideText: analysisBulletOverrides.overrideText,
        source: analysisBulletOverrides.source,
        suggestionId: analysisBulletOverrides.suggestionId,
      })
      .from(analysisBulletOverrides)
      .where(eq(analysisBulletOverrides.analysisId, analysisId))

    }
    ```

    Per DATA-05: The `where(eq(...analysisId))` clause ensures only overrides for the specific analysis are loaded, so two analyses on the same variant produce independent results.
