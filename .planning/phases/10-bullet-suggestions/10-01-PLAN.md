  **Actions:**
  - `accept(i)`: set state to 'accepted', clear editingIndex if matches
  - `dismiss(i)`: set state to 'dismissed', clear editingIndex if matches
  - `undo(i)`: set state to 'pending', restore finalText to original suggested_text
  - `acceptAll()`: set all pending suggestions to 'accepted'
  - `editFirst(i)`: set editingIndex = i (does not change state)
  - `updateFinalText(i, text)`: update finalText for suggestion at index i
  - `addSkill(i)`: set staged skill state to 'added'
  - `skipSkill(i)`: set staged skill state to 'skipped'
  - `undoSkill(i)`: revert staged skill to 'pending'
